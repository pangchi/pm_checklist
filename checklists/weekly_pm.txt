# Weekly Preventive Maintenance Checklist
# Frequency: Weekly | Est. Duration: 1–1.5 hours




#   SECTION: <Title>                 → creates a new section header
#   STEP: <Description>              → a checkbox step (no value input)
#   STEP_VALUE: <Description>        → a checkbox step WITH a required value input field
#   PHOTO: <Description>             → a checkbox step that requires a photo attachment
#   NOTE: <Text>                     → a read-only informational note shown to the user
#
# All steps within a section must be completed before moving to the next section.
# ─────────────────────────────────────────────────────────────────────────────

SECTION: Pre-Maintenance Safety Checks

NOTE: Ensure all safety gear (PPE) is worn before proceeding.
STEP: Confirm work order number matches equipment tag
STEP: Lock Out / Tag Out (LOTO) procedure applied
STEP: Area cordoned off and warning signs placed
STEP_VALUE: Record ambient temperature (°C)
STEP_VALUE: Record ambient humidity (%RH)

SECTION: Equipment Inspection

NOTE: Inspect all external components before opening panels.
STEP: Inspect exterior for physical damage or corrosion
STEP: Check all cable routing and strain reliefs
STEP_VALUE: Record equipment hour meter reading (hrs)
PHOTO: Photograph overall equipment condition before maintenance
STEP: Verify all access panels are accessible

SECTION: Mechanical Maintenance

STEP: Clean all external surfaces and vents
STEP_VALUE: Record fan speed (RPM) if applicable
STEP: Inspect belts and pulleys for wear
STEP: Lubricate all designated lubrication points
STEP_VALUE: Record lubricant type and quantity used (mL)
PHOTO: Photograph lubrication points after servicing

SECTION: Electrical & Controls Check

NOTE: Use calibrated instruments only.
STEP: Inspect terminal blocks for loose connections
STEP: Check fuses and circuit breakers — replace if blown
STEP_VALUE: Measure and record supply voltage (VAC)
STEP_VALUE: Measure and record operating current (A)
STEP: Verify all indicator lights function correctly
STEP: Test emergency stop (E-Stop) functionality [min=30]

SECTION: Functional Verification

STEP: Power on equipment and observe startup sequence [min=60]
STEP_VALUE: Record startup time (seconds)
STEP: Run equipment through full operating cycle [min=120]
STEP_VALUE: Record peak operating temperature (°C)
PHOTO: Photograph equipment in operating condition
STEP: Confirm all alarms and interlocks are active
STEP: Verify communication with MES/host system

SECTION: Post-Maintenance Sign-Off

NOTE: All steps above must be complete before sign-off.
STEP: Area cleaned and all tools accounted for
STEP: LOTO removed and equipment returned to service
STEP: Work order updated with findings and actions taken
STEP_VALUE: Record total maintenance duration (minutes)
PHOTO: Photograph final equipment condition after maintenance
